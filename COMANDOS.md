# Comandos para rodar local

Abre dois terminais, um para cada:

## Backend (API FastAPI — porta 8001)
```bash
cd apps/api
python main.py
```

## Frontend (Vite — porta 5174/5175/5176)
```bash
cd apps/web
npm run dev
```

O Vite imprime o link certinho no terminal, tipo `http://localhost:5174/`. Abre no navegador.
